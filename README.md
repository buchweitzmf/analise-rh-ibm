# 📊 Análise de RH — IBM HR Analytics

## Sobre o Projeto
Análise exploratória sobre o comportamento profissional de funcionários 
da IBM, com foco em identificar os fatores que influenciam o turnover 
(saída de funcionários da empresa).

## Ferramentas Utilizadas
- Python + Pandas
- Jupyter Notebook
- Power BI

## Perguntas Respondidas
- Qual departamento tem maior taxa de turnover?
- Funcionários que saem ganham menos do que os que ficam?
- Qual faixa etária concentra mais saídas?
- Satisfação no trabalho influencia o turnover?

## Principais Insights
- **Sales** é o departamento com maior turnover — 20% dos funcionários saem
- Funcionários que saem ganham em média **30% menos** do que os que ficam ($4.787 vs $6.832)
- A faixa etária **18-25 anos** concentra a maior taxa de saídas — 35%
- Funcionários com **baixa satisfação** têm o dobro de chance de sair — 23% vs 11%

## Dataset
Dataset disponível no Kaggle: [IBM HR Analytics Employee Attrition](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset)

## Como Executar
1. Clone o repositório
2. Abra o arquivo `IBM-Analytics.ipynb` no Jupyter
3. Execute as células em ordem
4. O dashboard está disponível em `IBM HR Analytics.pbix`
